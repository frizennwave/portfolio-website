# 🌐 Rifki Hanifudin — Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript. Features a clean dark-themed layout with sidebar navigation, animated skill bars, project showcase, blog section, and a contact form.

![Portfolio Preview](assets/img/profile.jpeg)

## ✨ Features

- **About Me** — Bio, services, testimonials, and client logos
- **Resume** — Education timeline, work experience, and skill bars
- **Portfolio** — Filterable project gallery (REST API, Applications, Other)
- **Blog** — Article cards with categories and read-time estimates
- **Contact** — Embedded map and contact form
- **Responsive Design** — Works on desktop and mobile devices
- **Dark Theme** — Modern dark UI with accent colors

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling & animations |
| JavaScript (Vanilla) | Interactivity & tab switching |
| Bootstrap 5.3 | Responsive grid & components |
| Bootstrap Icons | Icon set |
| Google Fonts (Syne, DM Sans) | Typography |

## 🚀 Getting Started

No installation needed. Just open `portfolio.html` in your browser:

```bash
# Clone the repository
git clone https://github.com/frizennwave/portfolio-website.git

# Navigate into the folder
cd portfolio-website

# Open in browser
open portfolio.html
```

Or simply drag and drop `portfolio.html` into any web browser.
